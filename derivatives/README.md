# Derivatives

This directory holds the output data files generated by `code/update.py`.

The output is available through two ephemeral branches:
- **`derivatives`**: the full output (`<cache_name>.jsonl`) tracked as a persistent DataLad dataset, with the complete history and provenance of every run.
- **`dist`**: the compressed consumer artifact (`<cache_name>.jsonl.gz`) produced by `code/compress.py`. This branch is force-recreated on every run and is what downstream users download (see the top-level `README.md`).
