# Derivatives

This directory holds the output data files generated by `code/update.py`.

On the `main` branch (this branch) it contains only this README — generated outputs are kept out of the code branch (see `.gitignore`). The actual outputs live on two other branches, both produced by `code/update_pipeline.sh`:

- **`derivatives` branch** — the full output (`<cache_name>.jsonl`) tracked as a persistent DataLad dataset, with the complete history and provenance of every run.
- **`dist` branch** — the compressed consumer artifact (`<cache_name>.jsonl.gz`) produced by `code/compress.py`. This branch is force-recreated on every run and is what downstream users download (see the top-level `README.md`).

Both branches also carry the study-level `dataset_description.json` (copied from `main` by `code/update_pipeline.sh`) so the published data is self-describing.
